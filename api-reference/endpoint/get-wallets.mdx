---
title: "Get Wallets"
api: "GET /api/v1/wallets"
description: "Retrieve all wallets for the authenticated workspace"
---

## Authentication

This endpoint requires authentication via Bearer token.

```bash
Authorization: Bearer YOUR_API_KEY
```

## Request

<ParamField query="workspace_id" type="string" required>
  The workspace ID to fetch wallets for
</ParamField>

<ParamField query="chain" type="string">
  Filter wallets by blockchain network
  - `ethereum`
  - `polygon`
  - `arbitrum`
  - `bsc`
</ParamField>

<ParamField query="page" type="number" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="number" default="50">
  Number of wallets per page (max 100)
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  <Expandable title="Wallet Object">
    <ResponseField name="id" type="string">
      Unique wallet identifier
    </ResponseField>

    <ResponseField name="address" type="string">
      Blockchain wallet address (0x...)
    </ResponseField>

    <ResponseField name="label" type="string">
      User-defined wallet label
    </ResponseField>

    <ResponseField name="chain" type="string">
      Blockchain network (ethereum, polygon, arbitrum, bsc)
    </ResponseField>

    <ResponseField name="balance" type="object">
      <Expandable title="Balance Details">
        <ResponseField name="native" type="string">
          Native token balance (ETH, MATIC, etc.)
        </ResponseField>

        <ResponseField name="tokens" type="array">
          Array of ERC-20 token balances
        </ResponseField>

        <ResponseField name="fiat_value" type="number">
          Total value in USD
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="transaction_count" type="number">
      Total number of transactions
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Wallet creation timestamp (ISO 8601)
    </ResponseField>

    <ResponseField name="last_synced" type="string">
      Last sync timestamp (ISO 8601)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="pagination" type="object">
  <Expandable title="Pagination Info">
    <ResponseField name="total" type="number">
      Total number of wallets
    </ResponseField>

    <ResponseField name="page" type="number">
      Current page number
    </ResponseField>

    <ResponseField name="pages" type="number">
      Total number of pages
    </ResponseField>

    <ResponseField name="limit" type="number">
      Results per page
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

<CodeGroup>

```bash cURL
curl -X GET "https://api.cryptotally.xyz/v1/wallets?workspace_id=ws_123&chain=ethereum" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch('https://api.cryptotally.xyz/v1/wallets?workspace_id=ws_123', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY'
}

params = {
    'workspace_id': 'ws_123',
    'chain': 'ethereum'
}

response = requests.get(
    'https://api.cryptotally.xyz/v1/wallets',
    headers=headers,
    params=params
)

data = response.json()
print(data)
```

```typescript TypeScript
interface Wallet {
  id: string;
  address: string;
  label: string;
  chain: string;
  balance: {
    native: string;
    tokens: Token[];
    fiat_value: number;
  };
  transaction_count: number;
  created_at: string;
  last_synced: string;
}

const getWallets = async (workspaceId: string): Promise<Wallet[]> => {
  const response = await fetch(`https://api.cryptotally.xyz/v1/wallets?workspace_id=${workspaceId}`, {
    headers: {
      'Authorization': `Bearer ${process.env.CRYPTOTALLY_API_KEY}`
    }
  });

  const { data } = await response.json();
  return data;
};
```

</CodeGroup>

## Rate Limits

- **Free Tier**: 100 requests per hour
- **Pro Tier**: 1,000 requests per hour
- **Enterprise Tier**: Custom limits

Rate limit headers are included in the response:

```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1679334400
```

## Filtering Examples

### Get Ethereum wallets only
```bash
GET /api/v1/wallets?workspace_id=ws_123&chain=ethereum
```

### Paginate results
```bash
GET /api/v1/wallets?workspace_id=ws_123&page=2&limit=20
```

## Webhook Support

Subscribe to wallet events:
- `wallet.created` - New wallet added
- `wallet.updated` - Wallet label or details changed
- `wallet.synced` - Wallet sync completed
- `wallet.deleted` - Wallet removed

See [Webhooks](/api-reference/webhooks) for setup instructions.
