## Overview

Learn how to export your transaction data for accounting and tax purposes.

<Note>
  This page is under construction. Content will be added as features are implemented.

## CSV Exports

Generate CSV files with:
- All transaction details
- Tags and notes
- Fiat values
- Filter by wallet, tag, or date range

## Use Cases

Perfect for:
- Tax preparation
- Accounting software import
- Grant auditors
- DAO treasury reports
- Quarterly/annual reporting

## Filtering

Export exactly what you need:
- By wallet
- By tag
- By date range
- By token
- Custom filters
