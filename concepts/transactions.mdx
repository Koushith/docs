## Auto-Sync

CryptoTally automatically fetches and syncs:
- Native token transfers (ETH, MATIC, etc.)
- ERC20 token transfers
- DEX swaps
- Historical transactions

## Fiat Values

- Automatic fiat conversion at time of transaction
- Powered by CoinGecko API
- Multi-currency support (USD, EUR, INR)
- Essential for tax reporting

## Transaction Details

Each transaction includes:
- Transaction hash
- From/To addresses
- Token and amount
- Fiat value at time of transaction
- Timestamp
- Transaction type (transfer, swap, etc.)
