## Overview

Track wallets across multiple blockchains with read-only access.

<Note>
  This page is under construction. Content will be added as features are implemented.
</Note>

## Key Features

- **Read-only access** - No private keys or transaction signing
- **Multi-chain support** - Ethereum, Polygon, Arbitrum, BNB Chain
- **Unlimited wallets** - Track as many wallets as you need (Pro+)
- **Wallet labeling** - Organize with custom labels
- **Consolidated view** - See all balances in one place
